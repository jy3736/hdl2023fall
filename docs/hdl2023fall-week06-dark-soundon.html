Combinational Logic Circuits - MUX and Arithmetic Circuits

組合邏輯電路 - 多工器與運算電路

A combinational logic circuit is a digital circuit that uses logic gates to manipulate input signals and generate output signals. The output depends only on the current input, as the circuit has no memory or feedback. Combinational logic circuits can perform various functions, such as data transmission, data encoding, data selection, arithmetic operations and calculations. Two examples of combinational logic circuits are multiplexers and arithmetic circuits. A multiplexer selects one of several input signals and forwards it to a single output line. An arithmetic circuit performs arithmetic operations on binary numbers. Both multiplexers and arithmetic circuits can be built using AND, OR and NOT gates.

This Week's Circuit Design Adventure

This week, you will learn how to make and use some simple logic circuits that can do things like choose between different inputs or add numbers. These circuits are called combinational logic circuits and they include multiplexers (MUX) and arithmetic circuits. You will see how you can build these circuits with logic gates, which are basic devices that can perform logical operations like AND, OR and NOT. You will also learn how to design and test these circuits using truth tables, Boolean algebra and HDLBits. You will practice with some examples of MUX and arithmetic circuits in class, such as a 4-to-1 MUX that can select one of four inputs, a half-adder that can add two bits, a full-adder that can add three bits and a 4-bit adder that can add four-bit numbers. After class, you will do more problems on HDLBits to improve your skills with combinational logic circuits.
